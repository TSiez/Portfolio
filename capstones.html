The Portfolio

Editorial personal site — Bystander Studio
Brief
A personal site that positions me as a Virtual Assistant and showcases my live builds — brand identity, layout, and front-end designed and hand-coded end to end, with no templates or page builders.
Stack
  • HTML · CSS · vanilla JavaScript (zero dependencies)
  • Claude — design, cowork & code
  • Vercel (deploy) · GitHub (version control)
  • FFmpeg — intro-reel compression
  • Type: Fraunces · Archivo · Space Mono
The process
  • Brand & identity — logo, type system, cream / navy / lime palette
  • Design system — editorial layout and reusable components
  • Hand-coded front-end — semantic HTML/CSS, no framework
  • Interaction & motion — folder toggles, deck galleries, scroll reveals
  • Ship — GitHub → Vercel, responsive with optimized media
Outcomelive · responsive · 0 deps
Built in~1 week

Project

The Lead List

B2B SaaS lead generation — case study
Brief
Briefed for 100 B2B SaaS founder leads with every paid prospecting tool blocked and no budget. I re-routed to a research-first workflow and shipped 43 verified, outreach-ready leads — for $0 in tooling.
Method
  • Manual web research from primary sources
  • Primary-source verification (live pages, public posts)
  • ICP screening & discipline
  • Email-pattern validation (first@domain)
  • Structured CSV delivery
  • Attempted: Apollo enrichment — blocked
The process
  • Source candidates from primary sources
  • Verify every name against a live page
  • Enforce the ICP — cut, don't pad the count
  • Recover emails from public signals
  • Ship a clean, structured CSV
Outcome43 leads · $0 tooling
Verified7 public-confirmed
